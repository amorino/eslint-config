# Next.JS
[![Version][version-badge]][package]

## Installation

```
npm i --dev eslint prettier @amorino/eslint-config-next
```
## Usage

Add to your eslint config (`.eslintrc`, or `eslintConfig` field in `package.json`):

```json
{
    "extends": "@amorino/eslint-config-next"
}
```
