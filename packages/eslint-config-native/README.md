# React Native
[![Version][version-badge]][package]

## Installation

```
npm i --dev eslint prettier @amorino/eslint-config-native
```

## Usage

Add to your eslint config (`.eslintrc`, or `eslintConfig` field in `package.json`):

```json
{
    "extends": "@amorino/eslint-config-native"
}
```
